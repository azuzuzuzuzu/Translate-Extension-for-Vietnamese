 # Translate Selection - Hướng dẫn sử dụng (Tiếng Việt)
 
 ## Tổng quan
 
 Translate Selection là tiện ích mở rộng cho Chrome giúp dịch nhanh đoạn văn bản bạn bôi đen trên trang web. Hỗ trợ chế độ tự động, popup nhập tay, phím tắt và nút FAB nổi.
 
 ## Cài đặt
 1. Clone hoặc tải mã nguồn
 2. Mở `chrome://extensions/`
 3. Bật Developer mode
 4. Click `Load unpacked` rồi chọn thư mục dự án
 
 ## Các chế độ sử dụng
 
 - **Popup**: Mở popup, nhập hoặc dán văn bản, chọn ngôn ngữ đích, nhấn `Dịch` hoặc Enter.
 - **Auto-Translate**: Bôi đen văn bản trên trang → popup hiện (nếu bật).
 - **Phím tắt**: Không có phím tắt mặc định (đã bỏ `Ctrl+Shift+T` để tránh xung đột với Chrome).
 - **FAB**: Nút nổi trên trang để dịch nhanh hoặc bật/tắt auto (giữ 0.6s).
 
 ## Lưu ý về ký tự
 - Giới hạn tối đa: 5.000 ký tự
 - Cảnh báo khi ≥ 4.500 ký tự
 
 ## Thao tác nhanh
 - `Enter` = Dịch (popup)
 - `Shift+Enter` = Xuống dòng
 - `Esc` = Đóng popup
 
 ## Xử lý lỗi phổ biến
 - "Không có kết nối Internet" → kiểm tra mạng
 - "Kết nối quá lâu" → thử lại sau hoặc tăng timeout
 - "Cần ít nhất 2 ký tự" → chọn nhiều ký tự hơn
 
 ## Mẹo
 - Dịch đoạn dài, chia nhỏ để tránh timeout
 - Bật chế độ tự động khi cần dịch nhiều đoạn nhỏ
 
 Cảm ơn bạn đã sử dụng tiện ích!
